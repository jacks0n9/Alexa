libsoup Reference Manual

libsoup API Reference

SoupAddress — Higher-level IP address object
SoupAuth — HTTP Authentication support
SoupConnection — a single possibly-persistent HTTP connection
SoupConnectionNTLM — NTLM-authenticated connection
SoupMessage — An HTTP request and response.
SoupMessageFilter — Automatic message processing
SoupServerMessage — Server-side SoupMessage
SoupServer — HTTP server
SoupSession — Soup session state object
SoupSessionAsync — Soup session for asynchronous (main-loop-based) I/O.
SoupSessionSync — Soup session for blocking I/O in multithreaded programs.
SoupSoapMessage — A SOAP request
SoupSoapResponse — A SOAP response
SoupSocket — a network socket
soup-status — HTTP and Soup status codes
soup-uri — URIs
SoupXmlrpcMessage — An XML-RPC Message
SoupXmlrpcResponse — An XML-RPC response
soup-misc — Miscellaneous functions
